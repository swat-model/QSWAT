Detailed Description

TIFF tag structure.

Attributes
----------
name : string
    Name of tag.
code : int
    Decimal code of tag.
dtype : str
    Datatype of tag data. One of TIFF DATA_FORMATS.
count : int
    Number of values.
value : various types
    Tag data as Python object.
ImageSourceData : int
    Location of value in file.

All attributes are read-only.
